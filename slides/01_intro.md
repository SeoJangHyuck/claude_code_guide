## 슬라이드 1-2 — 타임라인 (1분 30초)

**핵심 메시지**: "특별한 사람이 아니라, 환경 덕분에 일찍 써본 사람."

**본문 콘텐츠**:

```
2019 ──────── 2022 ──────── 2024~25 ──────── 오늘
   |             |              |               |
SI 개발자      크래프톤 입사   회사 차원의       부산연구원
(윈스테크넷)                AI 페어프로그래밍   (강연)
                            도입·권장 환경

  ──────────────────────────────────────────

  주요 경험
   • 통신사·공공 SI 프로젝트 (윈스테크넷, 2019~2022)
   • 사내 게임 데이터 대시보드 · BI 서비스 (2022~)
   • 글로벌 게임 공식 사이트 — 다국어(2~22개 언어) 대응
   • 글로벌 통합 유저 허브 플랫폼 (2026~)
```

**발표 노트 (1분 30초)**:
> "잠깐 제 이력만 보여드릴게요. 2019년에 SI 개발자로 시작해서 2022년에 크래프톤에 입사했습니다. 입사 후에는 사내 데이터 대시보드, 글로벌 게임 공식 사이트, 다국어 유저 허브 같은 프론트엔드 작업을 주로 했어요. 그러다 2024년 중반부터 회사가 AI 페어프로그래밍을 본격적으로 도입하는 분위기로 바뀌었고, 그 흐름 속에서 자연스럽게 Claude Code를 쓰게 됐습니다. 도구 변천사는 잠시 후 자세히 보여드릴게요. 제가 AI를 잘 알아서가 아니라, 회사 환경 덕분에 좀 일찍 접했을 뿐입니다."

**시각 표현 메모**:
- 상단: 가로 타임라인 1줄, 마디 4개. AI 도입·오늘 강조색.
- 하단: "주요 경험" 박스 — 4줄, 일반화된 분야 위주 (내부 코드명·구체 성과 수치 X)
- "업무에 ChatGPT → Cursor → Claude Code 순으로 도입" 부분은 챕터 3(변천사·하네스)에서 본격 다루므로 여기서는 제외

## 슬라이드 1-3 — 같은 회사원 메시지 (1분 30초)

**핵심 메시지**: 청중과 동일선상에 자기 자신을 놓는다.

**본문 콘텐츠**:

```
저는 AI 에이전트 전문가가 아닙니다.

여러분과 같은 회사원입니다.

오늘 1시간,
매일 쓰는 워크플로우를 그대로 보여드립니다.
```

**발표 노트 (1분 30초)**:
> "한 가지 분명히 하고 시작하겠습니다. 저는 AI 전문가가 아니에요. 에이전트 연구자도 아니고, 모델을 만드는 사람도 아닙니다. 여러분과 똑같이 매일 코드 짜고, 회의하고, 동료와 협업하는 회사원입니다. 오늘 1시간은 거창한 이론이 아니라 — 제가 매일 키보드 앞에서 쓰는 워크플로우를 그대로 펼쳐 보여드리는 시간입니다. 본격적으로 시작하기 전에, 잠깐 회사 이야기 한 장만 보고 가겠습니다."

**시각 표현 메모**:
- 4줄 텍스트만. 큰 글자, 줄간격 넓게.
- 마지막 줄로 자연스럽게 챕터 2(크래프톤 AI 정책)로 연결.
- 본인 사진은 표지(1-1)에만 작게 둘지 여기 한 번 더 둘지는 디자인 단계에서 결정.

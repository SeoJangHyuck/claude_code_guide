> 톤 가이드: 비개발자 청중 대상이라 "코드 검증" 표현은 "결과 검증"으로 일반화. 설문 1순위 환각 우려와 정확히 매핑. 한 슬라이드로 임팩트.

## 슬라이드 8-1 — superpowers: 검증을 강제하는 플러그인 (3분)

**핵심 메시지**: "AI가 빨리 일하다 설계·검증을 건너뛰는 문제를 강제로 멈추게 하는 플러그인. 설문 1순위 환각 우려의 정면 답."

**본문 콘텐츠**:

```
superpowers — 검증을 강제하는 플러그인

  ───────────────────────────────────────────

  AI가 빨리 일할수록 건너뛰기 쉬운 4가지를
  주요 단계마다 강제로 멈추고 챙기게 합니다.

  ┌────────────────────────────┬────────────────────────────┐
  │ 🧠 brainstorming           │ 🧪 test-driven-development │
  │ 만들기 전에 의도·요구사항  │ 구현 전에 검증 기준부터    │
  │ 명확화 강제                │ 정의                       │
  ├────────────────────────────┼────────────────────────────┤
  │ 🔍 systematic-debugging    │ ✅ verification-before-    │
  │ 즉시 수정 금지,            │    completion              │
  │ 원인 진단부터              │ 완료 선언 전 검증 체크리스트│
  └────────────────────────────┴────────────────────────────┘

  ───────────────────────────────────────────

  💡 왜 부산연구원 청중에게 추천하나
       설문 1순위 우려: AI 결과물 신뢰성·환각 (60%)
       superpowers의 핵심: 결과를 검증 없이 끝내지 못하게 강제
       → 설문 고민의 정면 답

  ───────────────────────────────────────────

  설치 (Anthropic 공식 마켓플레이스):
    /plugin install superpowers@claude-plugins-official

  또는 자체 마켓플레이스 (관련 플러그인 포함):
    /plugin marketplace add obra/superpowers-marketplace
    /plugin install superpowers@superpowers-marketplace

  📖 참고: github.com/obra/superpowers
```

**발표 노트 (3분)**:
> "마지막 추천 하나 드리고 마무리하겠습니다. superpowers라는 플러그인입니다. 제가 매일 쓰는 거예요.
>
> 한 줄 요약 — 'AI가 빨리 일하다 설계·검증을 건너뛰는 문제를 강제로 멈추게 하는 플러그인'입니다. AI는 답을 빨리 주고 싶어해요. 그래서 충분히 생각 안 하고 답하는 경우가 있는데, superpowers는 주요 단계마다 강제로 멈춥니다.
>
> 4가지 핵심이 있어요. brainstorming은 뭔가 만들기 전에 사용자 의도를 확실히 묻게 합니다. test-driven은 코드를 짜기 전에 '뭘로 검증할지' 먼저 정합니다. systematic-debugging은 '버그 보자마자 고치지 말고 먼저 원인 진단부터 해라'고 강제하고요. verification-before-completion은 '다 했다'고 선언하기 전에 체크리스트를 돌리게 합니다.
>
> 코드 위주처럼 보이지만, 핵심 가치는 비개발자 분께도 똑같이 맞아요. 오늘 설문에서 60%가 가장 걱정한 게 'AI 결과물 신뢰성·환각'이었죠? superpowers의 정체성이 정확히 그것입니다 — '검증 없이 끝내지 못하게 만드는 강제 게이트'. 즉 설문 1순위 고민의 정면 답이에요.
>
> 설치 방법은 두 가지인데 둘 다 명령 몇 줄이면 끝납니다. 한 번 깔아두시면 Claude Code가 알아서 단계마다 멈춰서 챙겨줍니다. 적극 추천드립니다.
>
> 그럼 마지막 정리로 넘어가겠습니다."

**시각 표현 메모**:
- 4분면 카드 그리드.
- 각 카드 이모지 + 한 줄 정의.
- 하단 💡 박스로 설문 매핑 강조 (이 슬라이드 핵심).
- 마지막 줄에서 챕터 9(마무리)로 자연 연결.

> 톤 가이드: 1시간 회고. 청중이 "내일 진짜 시작할 수 있다"는 확신을 가지고 강당을 나가도록.

---

## 슬라이드 9-1 — 내일 시작 3단계 (1분)

**핵심 메시지**: "지금 자리에서 일어나서 30분이면 1단계까지, 1시간이면 2단계까지 가능."

**본문 콘텐츠**:

```
내일부터 시작하는 3단계

  ───────────────────────────────────────────

  1️⃣  Claude Code 설치
       VSCode 확장 마켓에서 "Claude Code" 검색 → 설치
       소요: 5분

  2️⃣  내 업무 1개 골라 CLAUDE.md 작성
       매주 반복하는 작업 1개 선정 → 그 작업의 톤·규칙·디렉토리를
       CLAUDE.md 한 페이지로 정리
       소요: 30분

  3️⃣  그 업무를 skill로 묶고 옆자리에 공유
       2단계의 작업을 SKILL.md로 정리 → .claude/ 폴더 git commit
       (또는 zip으로 메신저 전달)
       소요: 1시간

  ───────────────────────────────────────────

  총 1시간 35분 = 이번 주 안에 다 가능합니다
```

**발표 노트 (1분)**:
> "오늘 들으신 모든 게 결국 이 3단계로 압축됩니다.
>
> 1단계, Claude Code 설치. VSCode 확장 마켓에서 검색해서 설치만 하면 5분이에요. 회사 컴퓨터 그대로, 새 IDE 안 깔아도 됩니다.
>
> 2단계, 매주 반복하는 작업 한 개만 골라서 CLAUDE.md를 작성해보세요. 30분이면 충분합니다.
>
> 3단계, 그 작업을 skill로 묶고 옆자리에 git url 또는 zip 한 번 공유. 1시간.
>
> 다 합쳐도 1시간 35분이에요. 이번 주 안에 다 가능합니다. 거창하게 생각하지 마시고, 오늘 자리 돌아가셔서 매주 반복하는 작업 하나만 떠올려 보세요. 거기가 출발점입니다.
>
> 그럼 마지막으로 Q&A 시간 가지겠습니다."

**시각 표현 메모**:
- 3단 카드.
- 각 단계 번호·소요 시간 강조.
- 마지막 줄("1시간 35분 = 이번 주 안") 강조.
- 마지막 멘트에서 챕터 10 (Q&A)으로 자연 연결.
